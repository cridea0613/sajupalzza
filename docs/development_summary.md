# SajuPalzza 개발 현황 요약 (Development Summary)

## 1. 프로젝트 개요

- **프로젝트명**: AI 사주분석 | 생년월일로 간단하게 보는 운세
- **서비스 URL**: [https://sajupalzza.pages.dev/](https://sajupalzza.pages.dev/)
- **목표**: AI를 활용해 현대적이고 신뢰도 높은 사주팔자 및 운세 분석 서비스를 제공합니다. 이를 통해 사용자가 자신의 삶에 대한 깊이 있는 통찰을 얻고, 긍정적인 방향으로 나아갈 수 있도록 돕는 것을 최종 목표로 합니다.
- **핵심 기능**: 사주 리포트, 토정비결, AI 궁합 추천, 리포트 저장/불러오기, 운세 기반 행동 지침 제공.

---

## 2. 최근 완료된 작업 (v1.2)

### 2.1. (신규) 리포트 저장 및 불러오기 기능

- **쿠키 기반 저장**: 사용자가 사주 분석 결과를 브라우저의 쿠키에 영구적으로 저장할 수 있는 기능을 추가했습니다. 이를 위해 `js-cookie` 라이브러리를 도입했습니다.
- **리포트 관리 UI**: 
    - 결과 페이지(`SajuResult.jsx`)에 '리포트 저장' 버튼을 추가하고, 이름을 지정할 수 있는 팝업창을 구현했습니다.
    - 초기 입력 페이지(`SajuForm.jsx`) 하단에 저장된 리포트 목록을 표시하는 `SavedReports.jsx` 컴포넌트를 새로 생성하고 연동했습니다.
- **사용자 편의 기능**: 저장된 목록에서 특정 리포트를 클릭하면 해당 정보가 입력 폼에 자동으로 채워지며, 개별 삭제 및 전체 초기화 기능도 함께 제공하여 관리의 편의성을 높였습니다.

### 2.2. 사용자 경험(UX) 개선

- **뒤로 가기 확인 기능**: 결과 페이지에서 사용자가 실수로 '처음으로' 버튼을 눌러도, 확인 팝업창을 통해 한 번 더 의사를 묻도록 개선했습니다. (`v1.1`)

### 2.3. 토정비결 기능 개선

- **렌더링 오류 해결 및 데이터 구조화**: `getTojeong` 함수 호출 오류를 수정하고, 결과를 총운, 행동 지침, 월별 운세로 명확하게 구조화했습니다. (`v1.0`)

### 2.4. UI/UX 개선

- **궁합/월별 운세 가독성 향상**: 궁합 추천 카드와 월별 운세 표의 시각적 디자인을 개선하여 정보 전달력을 강화했습니다. (`v1.0`)

## 3. 핵심 기능 명세

- **사주팔자 분석**: 생년월일시 기반의 만세력 계산 및 십신 분석 리포트 생성.
- **토정비결**: 연간 총운, 월별 운세, 그리고 구체적인 행동 지침 제공.
- **AI 궁합 추천**: 사용자의 일간(日干)을 기준으로 시너지가 좋은 궁합 추천.
- **사용자 편의 기능**:
    - **(신규) 리포트 저장/불러오기**: 분석 결과를 브라우저 쿠키에 저장하고, 언제든지 다시 불러오거나 관리할 수 있는 기능.
    - **뒤로 가기 확인**: 데이터 손실을 방지하기 위한 확인 팝업 기능.

## 6. 기술 스택 및 주요 라이브러리

- **프레임워크**: React (Vite 기반)
- **UI 라이브러리**: Material-UI (MUI)
- **쿠키 관리**: **(신규)** `js-cookie`
- **라우팅**: React Router
- **상태 관리**: React Hooks (`useState`, `useEffect`, `useMemo`), React Router (`useSearchParams`)
- **배포**: 정적 호스팅 (GitHub Pages 또는 유사 환경)
