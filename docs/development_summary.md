# SajuPalzza 개발 현황 요약 (Development Summary)

## 1. 프로젝트 개요

- **프로젝트명**: AI 사주분석 | 생년월일로 간단하게 보는 운세
- **서비스 URL**: [https://sajupalzza.pages.dev/](https://sajupalzza.pages.dev/)
- **목표**: AI를 활용해 현대적이고 신뢰도 높은 사주팔자 및 운세 분석 서비스를 제공합니다. 이를 통해 사용자가 자신의 삶에 대한 깊이 있는 통찰을 얻고, 긍정적인 방향으로 나아갈 수 있도록 돕는 것을 최종 목표로 합니다.
- **핵심 기능**: 사주 리포트, 토정비결, AI 궁합 추천, 운세 기반 행동 지침 제공.

---

## 2. 최근 완료된 작업 (v1.0)

### 2.1. 토정비결 기능 개선

- **렌더링 오류 해결**: `getTojeong` 함수 호출 시 누락되었던 연도(`currentYear`) 인자를 명확히 전달하여, 토정비결 데이터가 생성되지 않던 핵심 오류를 수정했습니다.
- **데이터 구조 재설계**: 토정비결 결과를 `general`(총운), `action`(행동 지침), `monthly`(월별 운세)로 구조화하여 데이터 처리의 명확성을 높였습니다.
- **컴포넌트 분리**: `TojeongResult.jsx` 컴포넌트를 생성하여 토정비결 UI 로직을 분리하고, `SajuResult.jsx`에서는 해당 컴포넌트를 호출하는 방식으로 구조를 단순화했습니다.

### 2.2. UI/UX 개선

- **궁합 추천 카드**: 각 궁합 타입(천생연분 등)에 고유한 배경색과 아이콘을 적용하고, 마우스 호버(hover) 시 그림자 효과를 추가하여 시각적 구분과 인터랙션을 강화했습니다.
- **월별 운세 가독성 향상**: 사용자의 제안에 따라, 기존의 단순 리스트 형태에서 벗어나 각 월을 시각적으로 강조하는 테이블 형식의 디자인(`MonthBox` 컴포넌트)을 도입하여 가독성을 크게 향상시켰습니다.

## 3. 핵심 기능 명세

- **사주팔자 분석**:
    - 사용자의 생년월일시 정보를 기반으로 만세력을 계산하여 사주팔자(연/월/일/시주)를 정확히 산출합니다.
    - 십신(十神) 관계를 분석하여 개인의 성향, 재물운, 직업운 등을 포함한 상세 리포트를 생성합니다.
- **토정비결**:
    - 연간 총운과 더불어, 구체적이고 실용적인 **행동 지침**을 함께 제공하여 사용자가 실생활에 적용할 수 있도록 돕습니다.
    - 12개월의 운세를 시각적으로 구분하여 제공합니다.
- **AI 궁합 추천**:
    - 사용자의 일간(日干)을 기준으로 오행의 상생 및 합(合) 원리에 따라 시너지가 좋은 궁합(일주 기준)을 추천합니다.

---

## 4. 디자인 가이드라인

- **전체 컨셉**: 모던하고 직관적인 인터페이스를 지향하며, 모바일 환경에 최적화된 반응형 디자인을 채택했습니다.
- **핵심 UI 컴포넌트**:
    - **궁합 카드**: 타입별 색상/아이콘 구분, 호버 효과를 통해 사용자 경험을 증진시킵니다.
    - **월별 운세**: `MonthBox` 컴포넌트를 사용하여 각 월의 정보를 시각적으로 강조하고, 운세 내용의 가독성을 확보했습니다.

## 5. 주요 페르소나 (From PRD)

- **가볍게 보는 사용자**: 무료 체험 후 신뢰가 형성되면 결제를 고려하는 사용자층.
- **고관여 사용자**: 사주표, 해석의 근거를 중요시하며 재방문 가능성이 높은 충성 고객층.
- **선물/공유 사용자**: 결과 리포트를 링크나 PDF로 저장하고 공유하고자 하는 니즈를 가진 사용자층.
