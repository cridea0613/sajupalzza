# SajuPalzza Blueprint

## 1. 개요

- **목표**: AI 기반의 현대적이고 신뢰도 높은 사주팔자 및 운세 분석 서비스를 제공하여, 사용자가 자신의 삶에 대한 통찰을 얻고 긍정적인 방향으로 나아갈 수 있도록 돕습니다.
- **핵심 기능**: 사주 리포트, 토정비결, AI 궁합 추천, 운세 기반 행동 지침 제공

## 2. 디자인 및 스타일 가이드

- **UI/UX**: 모던하고 직관적인 인터페이스. 모바일 우선 반응형 디자인. Inter 폰트 사용.
- **색상**: 주조색 (Primary), 강조색 (Accent), 배경색 (Background) 등을 정의하여 일관성 유지.
- **컴포넌트**: 재사용 가능한 컴포넌트(Button, Card, Modal 등)를 적극 활용하여 개발 효율성 증대.
- **궁합 추천 UI**: 각 궁합 타입(천생연분, 좋은 궁합, 친구 같은 궁합)에 따라 고유한 배경색과 아이콘을 사용하여 시각적 구분을 명확히 하고, 사용자가 마우스를 올렸을 때(hover) 그림자 효과를 주어 인터랙티브한 경험을 제공합니다. '자세히 보기' 버튼은 각 카드 우측 하단에 작고 깔끔하게 배치하여 공간 효율성과 시각적 균형을 맞춥니다.
- **월별 세운 UI**: 월별 운세를 표시할 때, 각 월을 나타내는 부분을 강조된 색상과 디자인(예: 그라데이션 배경, 그림자 효과)을 적용한 `MonthBox` 컴포넌트로 시각적으로 분리합니다. 이를 통해 사용자는 각 월의 운세 내용을 명확하게 인지할 수 있으며, 전체적인 가독성이 향상됩니다.

## 3. 기능 명세

### 3.1. 사주팔자 분석

- 사용자로부터 생년월일시 정보를 입력받아 만세력을 기반으로 사주팔자(연주, 월주, 일주, 시주)를 추출합니다.
- 십신(비견, 겁재, 식신, 상관, 편재, 정재, 편관, 정관, 편인, 정인)을 분석하여 개인의 성향, 재물운, 직업운 등을 해석한 리포트를 생성합니다.

### 3.2. 토정비결

- 사용자의 사주 정보와 현재 연도를 기반으로 연간 총운과 월별 운세를 계산합니다.
- **총운 행동 지침**: 연간 총운과 더불어, 사용자가 올해 어떤 행동을 취하면 좋을지에 대한 구체적이고 실용적인 조언을 함께 제공합니다.

### 3.3. AI 궁합 추천

- 사용자의 일간(日干)을 기준으로 가장 잘 맞는 궁합(천생연분, 좋은 궁합, 친구 같은 궁합)의 일주(日柱)를 추천합니다.
- 각 추천에 대한 상세한 설명과 해당 일주를 가진 사람의 성향 정보를 모달창으로 제공합니다.

### 3.4. 사용자 경험(UX) 개선

- **(신규) 뒤로 가기 확인 팝업**: 결과 페이지에서 사용자가 '처음으로' 버튼을 클릭할 경우, 바로 이동하지 않고 한 번 더 확인하는 팝업창을 표시합니다. 이를 통해 사용자가 실수로 입력 정보를 잃는 것을 방지합니다.
